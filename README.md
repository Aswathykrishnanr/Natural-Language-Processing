# Natural Language Processing (NLP) 

This repository contains **Natural Language Processing (NLP) experiments and projects** implemented using Python.  
It includes practical implementations such as **spam detection and Twitter sentiment analysis** using machine learning techniques.

---

## 📂 Repository Structure

```
Natural-Language-Processing/
│
├── Intro.ipynb                 # Introduction to NLP concepts
├── mail_spam.ipynb             # Email spam classification
├── twitter.ipynb               # Twitter sentiment analysis
├── SMS_train.csv               # Training dataset
├── SMS_test.csv                # Testing dataset
└── twitter_validation.csv      # Validation dataset
```

---

## 🔎 NLP Workflow

The following steps are explored in this repository:

- Text preprocessing
- Tokenization
- Stopword removal
- Feature extraction
- Text classification
- Spam detection
- Sentiment analysis
